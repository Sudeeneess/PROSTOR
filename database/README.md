**PROSTOR** — это проект архитектуры базы данных для современного маркетплейса. Система спроектирована с учетом высокой производительности, типизации данных и гибкого управления профилями пользователей.

---

## 🚀 Особенности архитектуры

Проект реализует комплексную логику интернет-магазина:
- **Умные профили (1:1):** Разделение базовой информации о пользователе и его роли (Покупатель, Продавец, Менеджер склада) без дублирования данных.
- **Финансовая точность:** Все цены и суммы используют тип `NUMERIC(12, 2)`, что исключает ошибки округления при расчетах.
- **Типизация через ENUM:** Статусы заказов, оплат и возвратов защищены на уровне базы данных.
- **Оптимизация (Индексы):** Настроены B-Tree индексы для быстрого поиска по товарам, категориям и истории заказов.
